# DINO의 뚝딱 블로그

사진과 짧은 메모만 입력하면, AI가 자연스러운 블로그 글을 뚝딱 만들어줍니다.

## 주요 기능

- **사진 업로드**: 최대 30장의 사진을 드래그 앤 드롭, 순서 변경 가능
- **글 스타일 선택**: 맛집 / 일상 / 여행 / 제품리뷰 / 홍보
- **톤/말투 선택**: 음슴체, 해요체, 합쇼체, 반말, 친근한 존댓말 + 자유 입력
- **AI 글 생성**: 사진을 분석하고 사람처럼 자연스러운 블로그 글 작성 (스트리밍)
- **에디터**: 사진 블록 단위로 직접 수정 가능
- **재생성**: 피드백을 주면서 마음에 들 때까지 재생성
- **미리보기**: 블로그 스타일로 완성된 글 확인
- **복사**: HTML / 텍스트 전체 복사 (사진 위치 placeholder 포함)

## 사용 방법

1. Claude API 키를 입력한다
2. 사진을 올린다 (선택)
3. 전체 주제 메모를 입력한다 (예: "강남 파스타 맛집, 분위기 좋았음")
4. 카테고리, 톤, 글 길이, 언어를 선택한다
5. AI가 블로그 글을 생성한다
6. 에디터에서 수정하고, 미리보기로 확인한다
7. 복사 버튼을 눌러 블로그에 붙여넣기한다

## 기술 스택

- **Frontend/Backend**: Next.js (React) + API Routes
- **AI**: Claude API (Anthropic) - 이미지 + 텍스트 멀티모달
- **배포**: Vercel

## 요구사항

- Node.js 18.17 이상
